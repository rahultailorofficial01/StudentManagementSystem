# studentresultmanagementsystem
🎓 Student Result Management System

A Python-based desktop application to manage student records with database integration and data visualization.

---

🚀 Features

- ➕ Add Student
- ❌ Delete Student
- 🔍 Search Student
- 🔄 Update Student
- 📋 View All Students
- 📊 Marks Visualization (Graph)

---

🛠️ Tech Stack

- Python
- SQLite (Database)
- Tkinter (GUI)
- Matplotlib (Graph)

---

📂 Project Structure

.
├── add_student.py
├── delete_student.py
├── update_student.py
├── search_student.py
├── view_students.py
├── marks_graph.py
├── students.db (auto-created)

---

▶️ How to Run

1. Install Python (3.x)
2. Install required library:

pip install matplotlib

3. Run any module:

python add_student.py

---

📊 Output

- GUI-based student management system
- Graph visualization of student marks
